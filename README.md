# VideoScan

Intelligent video analysis with AI-powered scene detection and object recognition.

## Features
- Automatic scene detection and segmentation
- Key frame extraction and annotation
- Object and activity recognition
- Temporal pattern analysis
- React + Tailwind SaaS webapp with Firebase Auth

## Quick Start
1. Clone this repo
2. Install: `pip install -e .`
3. Run: `videoscan --help`

## Web App
The `webapp/` directory contains a full SaaS frontend built with React + Tailwind CSS.
